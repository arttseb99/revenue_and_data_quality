# Проект 2: Анализ продаж и качества данных

## Описание
Проект посвящён анализу продаж компании с использованием SQL. 
Цель — определить наиболее прибыльные товары и категории, а также выявить проблемы в данных.

## Используемые инструменты
- SQL (JOIN, LEFT JOIN, CTE, агрегатные функции)
- Аналитическое мышление для бизнеса
- Проверка качества данных

## Основные результаты
- Выявлены товары с выручкой выше средней
- Определены ключевые категории по выручке
- Обнаружены ошибки в данных (NULL и некорректные product_id)
- Сформулированы рекомендации для бизнеса

- `project_2.sql` — SQL-запросы (анализ шагов 1–4)
